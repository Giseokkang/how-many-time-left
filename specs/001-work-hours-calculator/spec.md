---

## 목적

네이버 웍스 근무통계의 "일평균 잔여 시간"은 실제 의무 근로시간과 다르고, 금요일 재택(8h 고정)을 반영하지 않아 부정확하다.
실제 의무 근로시간 기준으로 **월~목 기준 일평균 몇 시간을 근무해야 하는지** 정확히 보여주는 크롬 익스텐션.

---

## 입력 / 출력

### 입력

| 필드 | 타입 | 소스 | 규칙 |
|------|------|------|------|
| 누적 근무시간 | HH:MM | 네이버 웍스 크롤링 | "산정 기간 내 누적 시간"에서 추출 (예: 114:09) |
| 해당 월 | YYYY-MM | 네이버 웍스 크롤링 | 페이지에 표시된 날짜에서 추출 |
| 공휴일 목록 | Date[] | holidays-kr JSON | 해당 월의 공휴일 자동 로드 |
| 수동 공휴일 | Date[] | 사용자 입력 (설정) | 임시공휴일 등 수동 추가/삭제 |
| 금요일 재택 제외 | boolean | 사용자 토글 | 기본값: true (제외) |

### 출력 (팝업 UI)

- 성공:
  - 이번 달 의무 근로시간 (예: 160h)
  - 현재 누적 근무시간 (예: 114h 09m)
  - 남은 근무시간 (예: 45h 51m)
  - 남은 근무일 수 (예: 6일)
  - **일평균 필요 근무시간** (예: 7h 38m) ← 핵심 정보
- 실패: "네이버 웍스 근무통계 페이지에서 열어주세요" 안내 메시지

---

## 계산 로직

```
1. 소정근로일 = 해당 월 평일 수 - 공휴일 수
2. 의무 근로시간 = 소정근로일 × 8h
3. 남은 총 근무시간 = 의무 근로시간 - 누적 근무시간

[금요일 재택 제외 ON일 때]
4. 남은 금요일 수 = 오늘 이후 남은 금요일 (공휴일 제외)
5. 남은 유동 근무시간 = 남은 총 근무시간 - (남은 금요일 × 8h)
6. 남은 유동 근무일 = 오늘 이후 남은 평일 - 공휴일 - 남은 금요일
7. 일평균 = 남은 유동 근무시간 ÷ 남은 유동 근무일

[금요일 재택 제외 OFF일 때]
4. 남은 근무일 = 오늘 이후 남은 평일 - 공휴일
5. 일평균 = 남은 총 근무시간 ÷ 남은 근무일
```

## 엣지케이스

- [ ] 네이버 웍스 페이지가 아닌 곳에서 팝업 열었을 때
- [ ] 월초(근무 기록 없음)에 누적시간이 0일 때
- [ ] 남은 근무일이 0일 때 (월말 마지막 근무일 이후)
- [ ] 이미 의무 근로시간을 초과했을 때 (남은 시간이 음수)
- [ ] 오늘이 금요일일 때 오늘을 남은 일수에 포함할지
- [ ] 공휴일이 금요일과 겹칠 때

---

## 제약사항

- 사용할 라이브러리: holidays-kr (공휴일 JSON 데이터)
- 대상 페이지: `workplace.worksmobile.com/my-space/work-statistics`
- Out of Scope:
  - 최대 근로시간(170h) / 연장근무 계산
  - 출퇴근 기록 상세 조회
  - 연차/반차/반반차 별도 관리
  - 여러 달 통계/히스토리
