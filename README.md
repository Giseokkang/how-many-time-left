# 근무시간 계산기 (Chrome Extension)

네이버 웍스 홈 화면에서 **실제 일평균 필요 근무시간**을 계산해주는 크롬 익스텐션입니다.

네이버 웍스의 "일평균 잔여 시간"은 실제 의무 근로시간과 다를 수 있고, 금요일 재택(8h 고정) 등을 반영하지 않습니다. 이 익스텐션은 공휴일 데이터와 실제 근무일 기준으로 정확한 일평균을 계산합니다.

## 주요 기능

- 네이버 웍스 홈 화면에서 누적 근무시간 자동 추출
- 공휴일 반영한 의무 근로시간 자동 계산
- 금요일 재택(8h 고정) 제외 옵션
- 수동 공휴일 추가/삭제 (임시공휴일 등)
- 설정값 자동 저장

## 설치 방법

1. 이 저장소를 클론하거나 [zip으로 다운로드](../../archive/refs/heads/main.zip)합니다
2. 크롬 주소창에 `chrome://extensions` 입력
3. 우측 상단 **"개발자 모드"** ON
4. **"압축해제된 확장 프로그램을 로드합니다"** 클릭 → 다운로드한 폴더 선택

## 사용 방법

1. [네이버 웍스 홈](https://home.worksmobile.com/) 접속
2. "나의 근로 시간" 위젯이 보이는 상태에서 익스텐션 아이콘 클릭
3. 일평균 필요 근무시간 확인

## 기술 스택

- Chrome Extension (Manifest V3)
- Vanilla JavaScript (빌드 스텝 없음)
- 공휴일 데이터: [holidays-kr](https://github.com/nicebb/holidays-kr)
